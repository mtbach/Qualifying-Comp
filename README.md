# Qualifying Fastest Lap Comparison Between Drivers 4 and 81 in 2025 Season.

This dashboard showcases an interactive circuit graph with side-by-side comparison of metrics and car behavior between two drivers for each qualifying session of the season.\
Car telemetry and location data is pulled from OpenF1 API. Data is warehoused on **Snowflake** and modeled with **dbt**. Visualizations and analytics are performed with **Python**, via **Plotly Dash**.
